#### &lt;hype&gt;

Tendance / alternative à NPM 

- Installer les dépendances de mon projet
```
$ cd monprojet
$ yarn
```

- Ajouter une dépendance à mon projet
```
$ yarn add lodash
```

- Ajouter une dépendance de dev à mon projet
```
$ yarn add eslint --dev
```
